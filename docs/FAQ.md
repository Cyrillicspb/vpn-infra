# FAQ

## У меня CGNAT. Можно ли установить?

Нет. CGNAT означает что вы делите внешний IP с другими пользователями.
Port forwarding невозможен.

**Решения**:
1. Попросить провайдера дать реальный IP (часто бесплатно или за небольшую доплату)
2. Перевести роутер в режим bridge mode (если двойной NAT)
3. Использовать VPS как основной сервер (без домашнего)

## VPN периодически отваливается

Увеличьте UDP NAT timeout на роутере до 180+ секунд.

## После /vpn add домен не работает

DNS кэш клиента обновляется до 5 минут. Подождите или очистите DNS кэш:
- Windows: `ipconfig /flushdns`
- macOS: `sudo dscacheutil -flushcache`
- Android/iOS: переключите WiFi

## QR-код не генерируется

Если AllowedIPs > 50 записей — QR не влезает. Используйте .conf файл.

## Я за границей, VPN нужен?

Неоптимально — вы добавляете лишний hop через домашний сервер в России.
Для зарубежного использования отключите VPN или настройте исключения.

## Как отключить Private DNS (DoH/DoT) на Android?

Настройки → Сеть и интернет → Расширенные настройки → Приватный DNS → Выключить.

Иначе dnsmasq не увидит DNS-запросы и не добавит IP в blocked_dynamic.

## Можно ли использовать с офисным VPN?

Да, но требуется настройка исключений. Используйте `/exclude add <офисная_подсеть>`.
Наши подсети (10.177.x.x) не пересекаются с типичными офисными.

## do-release-upgrade сломал систему

`do-release-upgrade` запрещён (удаляет DKMS модули). Обновление ОС только через
переустановку + restore.sh.

## Как добавить новый заблокированный сайт моментально?

```
/vpn add example.com
```

Изменения применяются немедленно. Обновление баз РКН происходит автоматически в 03:00.

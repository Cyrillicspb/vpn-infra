# Требования

## Минимальные требования

### Домашний сервер
- x86_64, 2+ ядра, 4 GB RAM, 64 GB SSD
- Ubuntu Server 24.04 LTS
- Ethernet (не Wi-Fi для сервера)
- Реальный (белый) IP или DDNS

### VPS
- KVM виртуализация (не OpenVZ)
- 2 vCPU, 2 GB RAM, 40 GB SSD
- 1+ TB трафика в месяц
- Порты TCP/UDP 443 открыты

### Роутер
- Поддержка port forwarding UDP
- Реальный IP (не CGNAT)

## Внешние зависимости

**Обязательные**:
- VPS (один или несколько)
- Telegram (для бота и алертов)
- Реальный IP

**Опциональные**:
- Cloudflare аккаунт (бесплатный, для CDN-стека)
- Домен (для DDNS через Cloudflare)

## Расчёт пропускной способности

Формула: `max_clients ≈ upload_Mbps ÷ 5`

Для HD (1080p): ~5 Мбит/с на клиента
Для 4K: ~20 Мбит/с на клиента

| Upload | HD клиенты | 4K клиенты |
|--------|-----------|------------|
| 10 Мбит/с | 2 | 0 |
| 50 Мбит/с | 10 | 2 |
| 100 Мбит/с | 20 | 5 |
| 500 Мбит/с | 100 | 25 |

## Провайдеры VPS

Проверенные (поддерживают порт 443 UDP/TCP):
- Hetzner (Германия/Финляндия) — дешево
- DigitalOcean (Амстердам/Нью-Йорк) — стабильно
- Vultr — гибко
- Linode/Akamai — надёжно

**Избегать**: провайдеры с UDP ограничениями или OpenVZ.

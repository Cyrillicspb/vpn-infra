# Оборудование

## Домашний сервер

| Параметр | Минимум | Рекомендуется |
|----------|---------|---------------|
| CPU | x86_64, 2 ядра | 4+ ядра |
| RAM | 4 GB | 8+ GB |
| SSD | 64 GB | 128+ GB |
| Сеть | Ethernet | Ethernet |
| ОС | Ubuntu 24.04 | Ubuntu 24.04 |

**Формула max_clients**: `upload_Mbps ÷ 5` (для HD видео)

Примеры:
- 10 Мбит/с upload → ~2 клиента
- 50 Мбит/с upload → ~10 клиентов
- 100 Мбит/с upload → ~20 клиентов

## VPS

| Параметр | Минимум |
|----------|---------|
| Тип | KVM (не OpenVZ!) |
| CPU | 2 vCPU |
| RAM | 2 GB |
| SSD | 40 GB |
| Трафик | 1+ TB/мес |

**Рекомендуемые провайдеры**: Hetzner, DigitalOcean, Vultr, Linode

**Рекомендуемые локации**: US, Germany, Netherlands, Finland

## Роутер

- Поддержка port forwarding
- Реальный (белый) IP — **обязательно**
- Или DDNS при динамическом IP

**Проверка CGNAT**: если внешний IP начинается с 100.64–100.127 — CGNAT.
Решение: попросить провайдера дать реальный IP (часто бесплатно).

## Источник бесперебойного питания (UPS)

Рекомендуется для домашнего сервера — при отключении электричества
VPN-клиенты потеряют доступ к заблокированным ресурсам.
